
# Анализ данных сервиса проката самокатов GoFast

## Описание проекта

В этом проекте я выступаю в роли аналитика компании GoFast — сервиса аренды электросамокатов.  
Цель — изучить демографию пользователей, особенности поездок, оценить выгоду от платной подписки и проверить статистические гипотезы.

Исходные данные:
- **Пользователи** (`users_go.csv`) — ID, имя, возраст, город, тип подписки (free / ultra).
- **Поездки** (`rides_go.csv`) — ID пользователя, расстояние (м), длительность (мин), дата.
- **Подписки** (`subscriptions_go.csv`) — стоимость минуты, старта поездки и ежемесячный платёж для каждого типа подписки.

## Этапы работы

1. **Загрузка и первичный осмотр данных**  
   - Импорт pandas, чтение трёх CSV-файлов.
   - Проверка количества строк, типов данных.

2. **Предобработка**  
   - Преобразование дат в `datetime`, выделение месяца.
   - Удаление дубликатов, заполнение пропусков.
   - Округление длительности поездки до целых минут.

3. **Исследовательский анализ (EDA)**  
   - Распределение пользователей по городам и типам подписки.
   - Визуализация: круговая диаграмма подписок, гистограмма возрастов.
   - Выявление доли несовершеннолетних пользователей.
   - Статистика длительности поездок (среднее, процентили).

4. **Объединение данных**  
   - Слияние таблиц в единый датафрейм `df`.
   - Выделение подвыборок `df_ultra` и `df_free`.

5. **Агрегация и расчёт выручки**  
   - Группировка по пользователю и месяцу.
   - Расчёт суммарного расстояния, времени, числа поездок.
   - Создание функции для вычисления месячной выручки:  
     `выручка = старт_поездки × кол-во + цена_минуты × сумма_минут + абонентская_плата`
   - Поиск пользователя с максимальной выручкой.

6. **Проверка гипотез (статистические тесты)**  
   - **H1:** Средняя длительность поездки у пользователей с подпиской больше, чем без неё.  
     → Односторонний t-тест для независимых выборок.
   - **H2:** Средняя дистанция поездки у подписчиков превышает 3130 м.  
     → Односторонний t-тест относительно заданного значения.
   - **H3:** Средняя месячная выручка от подписчиков выше, чем от пользователей без подписки.  
     → Односторонний t-тест для независимых выборок по агрегированным данным.

7. **Моделирование распределений (CDF / PPF)**  
   - Оценка вероятности поездок дольше 30 минут для подписчиков.
   - Вероятность попадания длительности в интервал 20–30 минут.
   - Расчёт 90-го процентиля дистанции для всех пользователей (критическое значение).
