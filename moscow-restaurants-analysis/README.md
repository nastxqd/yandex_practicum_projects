# Исследование рынка заведений общественного питания Москвы

## 📌 Цель проекта
Провести комплексное исследование рынка общественного питания Москвы для определения оптимальных параметров открытия нового заведения, которое будет максимально соответствовать рыночным трендам и предпочтениям потребителей.

### Задачи проекта:
1. Изучить структуру рынка по категориям заведений
2. Проанализировать географическое распределение заведений по районам Москвы
3. Исследовать соотношение сетевых и несетевых заведений
4. Определить типичную вместимость (количество мест) для разных категорий
5. Проанализировать рейтинги заведений и факторы, влияющие на них
6. Изучить ценовую политику и средние чеки в разных районах
7. Выявить популярные сети и их особенности
8. Проанализировать режим работы заведений (круглосуточные и другие)

---

## 📁 Данные

### Основные данные о заведениях (`rest_info.csv`):
- `name` — название заведения
- `address` — адрес заведения
- `district` — административный район (например, Центральный административный округ)
- `category` — категория заведения (кафе, пиццерия, кофейня и др.)
- `hours` — информация о днях и часах работы
- `rating` — рейтинг заведения по оценкам пользователей Яндекс Карт (шкала от 0 до 5.0)
- `chain` — признак сетевого заведения:
  - 0 — не сетевое
  - 1 — сетевое
- `seats` — количество посадочных мест

### Данные о ценах (`rest_price.csv`):
- `price` — категория цен (средние, ниже среднего, выше среднего и т.д.)
- `avg_bill` — строковое описание средней стоимости заказа (диапазоны цен)
- `middle_avg_bill` — числовая оценка среднего чека (только для записей, начинающихся с "Средний счёт")
- `middle_coffee_cup` — числовая оценка стоимости чашки капучино (только для записей, начинающихся с "Цена одной чашки капучино")

## ⚙️ Этапы проекта

### 1. Загрузка и предобработка данных
- Первичная проверка данных на соответствие типов
- Обработка пропусков и выбросов
- Заполнение пропусков:
  - `hours` (6%) — заполнены значением "режим работы не указан"
  - `price` и `avg_bill` — пропуски заменены на "не указано"
  - `seats` (43%) — заполнены медианными значениями по комбинации категория-район
- Удаление дубликатов (2 записи, 0.0% от общего объёма)
- Нормализация текстовых данных (`name`, `address`, `district`)

### 2. Генерация признаков
- Создание столбца `is_24_7` — признак круглосуточной работы заведения
- Оптимизация типов данных (`chain` → `int8`, `seats` → `int16`)

### 3. Исследовательский анализ
- Анализ структуры рынка по категориям
- Географическое распределение заведений
- Анализ сетевых и несетевых заведений
- Исследование вместимости заведений
- Анализ рейтингов и факторов влияния
- Корреляционный анализ
- Исследование топ-15 сетей
- Анализ ценовой политики по районам

## 📊 Результаты

### Структура рынка
✅ **Доминирующие категории**:
- Рестораны — 30% рынка
- Кафе — 20% рынка
- Кофейни — 19% рынка
- Топ-3 категории занимают 69% всех заведений

✅ **Соотношение сетевых и несетевых**:
- Несетевые заведения — 62%
- Сетевые заведения — 38%
- Наибольшая сетевизация: булочные (61,1%), пиццерии (52,1%), кофейни (51%)

### Географическое распределение
✅ **Лидеры по количеству заведений**:
- ЦАО — 2 242 заведения (26.7% от всех)
- САО — 931 заведение
- ЮАО — 922 заведения
- СЗАО — наименьшее количество (409 заведений)

✅ **Структура ЦАО**:
- Рестораны — 29.9%
- Кафе — 19.4%
- Кофейни — 19.2%

### Вместимость заведений
✅ **Типичное количество мест**:
- Бары и рестораны — медиана 111 мест
- Кафе — медиана 80 мест
- Кофейни — медиана 60 мест
- Быстрое питание — медиана 50 мест

✅ **Аномалии**:
- Максимум: 1 288 мест (банкетные залы, фуд-корты)
- Минимум: 0-4 места (форматы "навынос")

### Рейтинги заведений
✅ **Средние рейтинги по категориям**:
- Бары и пабы — 4.4 (максимальный)
- Булочные — 4.3
- Кафе — 4.2
- Быстрое питание — 4.1 (минимальный)

✅ **Корреляционный анализ**:
- Самая сильная связь с рейтингом — ценовая категория (0.28)
- Остальные факторы имеют слабую корреляцию

### Топ-15 сетей Москвы

✅ **Лидеры по качеству**:

- Кулинарная лавка — рейтинг 4.39

### Ценовая политика

- ЦАО — 1 191 руб. (максимальный)
- ЗАО — 1 053 руб.
- ЮВАО — 654 руб. (минимальный)

## 💡 Рекомендации для инвесторов

### 1. Выбор локации
- **Оптимальные округа**: САО, ЮАО, СВАО
  - Хорошая плотность заведений
  - Умеренная конкуренция
  - Сбалансированные цены
- **Избегать**: ЦАО (высокая конкуренция, высокие затраты)
- **Рассмотреть**: ЗАО для премиального формата

### 2. Выбор формата
- **Наиболее перспективно**:
  - Кофейня (высокий спрос, умеренные инвестиции)
  - Специализированное кафе
- **Высокий потенциал**:
  - Булочные/пекарни (высокие рейтинги, лояльная аудитория)
- **Осторожно**:
  - Рестораны (высокие затраты, сложность поддержания рейтинга)

### 3. Операционные параметры
- **Вместимость**: 80-100 мест (оптимальная загрузка)
- **Формат**: независимое заведение (больше гибкости)
- **Фокус**: качество обслуживания для поддержания рейтинга выше 4.5
- **Ценовая политика**: позиционирование в сегменте "выше среднего" для компенсации затрат

### 4. Стратегические рекомендации
- Учитывать высокую концентрацию заведений в центре при выборе локации
- Ориентироваться на форматы с высокими рейтингами (бары, булочные)
- При запуске сети начинать с 3-5 точек для контроля качества
- Инвестировать в поддержание высокого рейтинга (ключевой фактор успеха)
