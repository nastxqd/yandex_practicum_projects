# Исследование стартапов: анализ инвестиций и динамики финансирования

## 📌 Цель проекта
Подготовить датасет к работе, исследовать динамику и структуру финансирования стартапов и ответить на вопросы, важные для оценки инвестиционных стратегий.

### Задачи проекта:
1. Провести предобработку данных, убрать дубликаты и пропуски, проверить корректность числовых и временных значений
2. Выделить группы компаний по срокам финансирования и сравнить их по количеству и объёму инвестиций
3. Классифицировать сегменты рынка на массовые, средние и нишевые и учесть это в дальнейшем анализе
4. Определить типичные и аномальные значения объёмов финансирования, исключить выбросы и ограничить период исследования
5. Сравнить популярность и объёмы разных типов финансирования
6. Проанализировать динамику раундов и объёмов инвестиций по годам, а также изменения в массовых сегментах рынка
7. Рассчитать долю возврата средств для разных типов финансирования и оценить её устойчивость
8. Подвести итоговые выводы и дать рекомендации для инвестирования

## 📁 Данные

### Основной датасет — `cb_investments.zip`
Содержит информацию о компаниях и их финансировании:

| Поле | Описание |
|------|----------|
| `name` | Название компании |
| `market` | Основной рынок или отрасль компании |
| `funding_total_usd` | Общий объём привлечённых инвестиций в долларах США |
| `status` | Текущий статус компании (operating, closed и др.) |
| `country_code` | Код страны |
| `funding_rounds` | Общее число раундов финансирования |
| `first_funding_at` | Дата первого финансирования |
| `mid_funding_at` | Дата среднего по времени раунда финансирования |
| `last_funding_at` | Дата последнего финансирования |
| `seed`, `venture`, `angel` и др. | Суммы инвестиций по типам финансирования |

### Дополнительный датасет — `cb_returns.csv`
Содержит суммы возвратов по типам финансирования в миллионах долларов по годам:

- `year` — год возврата средств
- `seed`, `venture`, `debt_financing`, `angel` и др. — суммы возвратов по соответствующим типам

## ⚙️ Этапы проекта

### Шаг 1. Загрузка и предобработка данных
- Приведение названий столбцов к единому стилю
- Преобразование `funding_total_usd` в числовой тип (удаление разделителей разрядов)
- Конвертация дат в формат datetime
- Заполнение пропусков в текстовых столбцах заглушками
- Удаление полных дубликатов (4 855 записей)
- Удаление строк без данных о финансировании (8 536 записей)
- Заполнение `mid_funding_at` серединой интервала между `first_funding_at` и `last_funding_at`

### Шаг 2. Инжиниринг признаков
#### 2.1 Группы по срокам финансирования
- **Единичное финансирование** — один раунд
- **Срок до года** — между первым и последним раундом ≤ 365 дней
- **Срок более года** — финансирование длится более года

#### 2.2 Классификация сегментов рынка
- **Массовые** — более 120 компаний в сегменте
- **Средние** — от 35 до 120 компаний
- **Нишевые** — менее 35 компаний

### Шаг 3. Работа с выбросами и анализ
- Определение типичного диапазона финансирования (IQR метод)
- Выявление аномальных компаний по каждому сегменту
- Фильтрация данных за период с достаточным количеством раундов (≥50 в год)
- Анализ популярности и объёмов различных типов финансирования

### Шаг 4. Анализ динамики
- Расчёт среднего объёма одного раунда финансирования
- Анализ динамики размера раундов и количества раундов по годам
- Выявление массовых сегментов с ростом финансирования в 2014 году
- Расчёт нормированных значений возврата средств по типам финансирования

### Шаг 5. Итоговый вывод и рекомендации
- Формулирование рекомендаций для инвестирования на 2015 год

## 📈 Ключевые выводы

### Структура рынка
✅ **Преобладание единичного финансирования** — 59% компаний имеют только один раунд, но они привлекают лишь 31% объёма инвестиций

✅ **Долгосрочное финансирование** — компании с финансированием более года составляют 30% от общего числа, но получают 62% всех инвестиций

### Выбросы и аномалии
✅ **Нишевые сегменты более рискованны** — 15.0% компаний в нишевых сегментах получают аномальное финансирование (против 13.0% в массовых)

✅ **Real Estate лидирует по волатильности** — наибольшая доля выбросов среди массовых сегментов (17.9%)

### Типы финансирования
✅ **Венчурное финансирование** — абсолютный лидер (76% всех возвратов, 105 млрд USD привлечено)

✅ **Посевные инвестиции** — наиболее доступный инструмент (17 454 компании)

✅ **Альтернативные методы** (краудфандинг, конвертируемые займы) остаются нишевыми

### Динамика
✅ **Снижение размера раундов после 2005 года** — с $4.8 млн до $0.6 млн в 2014 году

✅ **Сокращение количества раундов в 2014 году** — снижение на треть по сравнению с 2013 годом

✅ **Manufacturing — самый быстрорастущий сегмент** — рост финансирования на 22.7% в 2014 году

## 💡 Рекомендации для инвестирования (на 2015 год)

### 1. Рекомендуемая отрасль: **Manufacturing**

**Обоснование:**
- Самый высокий абсолютный объём финансирования среди растущих сегментов
- Устойчивая положительная динамика на протяжении нескольких лет
- Значительный рост в 2014 году (+22.7% по сравнению с 2013 годом)
- Несмотря на периодические коррекции, сегмент демонстрирует уверенный восходящий тренд

### 2. Рекомендуемый тип финансирования: **Венчурное финансирование (Venture)**

**Обоснование:**
- Наиболее устойчивый рост относительной эффективности возвратов
- Абсолютный лидер по объёму привлечённых средств (105 млрд USD)
- Стабильная динамика возвратов с минимальной волатильностью
- Развитая инфраструктура и прозрачные механизмы работы
- Генерирует 76% всех возвратов от инвестиций
