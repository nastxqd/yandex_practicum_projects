# yandex_practicum_projects
Projects for Yandex Data Analyst Course

## 📌 Цель проекта
На основании данных сервиса Яндекс.Недвижимость необходимо провести исследовательский анализ данных и установить параметры, которые влияют на цену объектов. В дальнейшем это позволит построить автоматизированную систему для отслеживания аномалий и мошеннической деятельности.

## 📁 Данные
Архив объявлений о продаже квартир в Санкт-Петербурге и Ленинградской области за несколько лет. По каждому объекту доступно два вида данных:
- пользовательские (площадь, высота потолков и т.п.)
- автоматические на основе картографических данных (расстояние до центра и других объектов, кол-во парков и водоемов)

## ⚙️ Этапы проекта
1. **Загрузка и предобработка данных:** первичная проверка данных на соответствие типов, обработка пропусков и выбросов.
2. **Генерация признаков:** добавление дополнительных признаков для удобства дальнейшего анализа.
3. **Исследовательский анализ:** изучение распределений и долей категорий, корреляционный анализ и поиск закономерностей.
4. **Выводы и рекомендации:** итоговые выводы о найденных закономерностях в данных.

## 📊 Результаты
В ходе работы изучены ключевые факторы, влияющие на стоимость квартир. Получена информация о среднестатистическом предложении на рынке, выделены ТОП-10 населенных пунктов по активности рынка недвижимости и стоимости объектов.
