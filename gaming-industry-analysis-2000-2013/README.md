# Эволюция игровой индустрии (2000-2013) — путь к современным RPG

## 📌 Цель проекта
Провести комплексный анализ исторических данных игровой индустрии периода 2000-2013 годов для создания статьи, которая привлечет внимание любителей старых игр и повысит интерес к RPG-проекту "Секреты Темнолесья".

### Задачи проекта:
1. Изучить развитие игровой индустрии с 2000 по 2013 год
2. Проанализировать продажи игр на различных платформах
3. Исследовать жанровые предпочтения игроков в разных регионах
4. Провести категоризацию игр по оценкам пользователей и критиков
5. Выделить ключевые платформы периода
6. Подготовить данные для создания статьи об индустрии

---

## 📁 Данные

Данные `/datasets/new_games.csv` содержат информацию о продажах игр разных жанров и платформ, а также пользовательские и экспертные оценки игр:

- `Name` — название игры
- `Platform` — название платформы
- `Year of Release` — год выпуска игры
- `Genre` — жанр игры
- `NA sales` — продажи в Северной Америке (в миллионах проданных копий)
- `EU sales` — продажи в Европе (в миллионах проданных копий)
- `JP sales` — продажи в Японии (в миллионах проданных копий)
- `Other sales` — продажи в других странах (в миллионах проданных копий)
- `Critic Score` — оценка критиков (от 0 до 100)
- `User Score` — оценка пользователей (от 0 до 10)
- `Rating` — рейтинг организации ESRB (Entertainment Software Rating Board)

## ⚙️ Этапы проекта

### 1. Загрузка и знакомство с данными
- Первичная проверка данных на соответствие типов
- Выявление особенностей структуры датасета
- Анализ объёма данных (16 956 записей)

### 2. Предобработка данных
#### 2.1 Приведение названий столбцов к snake_case
- Преобразование всех названий в нижний регистр
- Замена пробелов на подчёркивания

#### 2.2 Корректировка типов данных
- Преобразование `eu_sales`, `jp_sales`, `user_score` из object в float64
- Замена строковых значений (`unknown`, `tbd`, `n/a`) на пропуски

#### 2.3 Обработка пропусков
- Создание вспомогательных столбцов `has_critic_score`, `has_user_score`
- Заполнение пропусков в `rating` значением `NO_RATING`
- Заполнение пропусков в продажах средними значениями по группам (год + платформа)

#### 2.4 Удаление дубликатов
- Нормализация текстовых данных (названия игр → верхний регистр, жанры → нижний регистр)
- Исправление неявных дубликатов в рейтинге (`K-A` → `E`)
- Удаление явных дубликатов (241 запись)

### 3. Фильтрация данных
- Отбор игр за период 2000-2013 годы включительно
- Формирование рабочего датафрейма `df_actual` (13 050 записей)

### 4. Категоризация данных
#### Пользовательские оценки (User Score, шкала 0-10):
- **Высокая оценка** — от 8 до 10 (включительно)
- **Средняя оценка** — от 3 до 8 (не включая 8)
- **Низкая оценка** — от 0 до 3 (не включая 3)

#### Оценки критиков (Critic Score, шкала 0-100):
- **Высокая оценка** — от 80 до 100 (включительно)
- **Средняя оценка** — от 30 до 80 (не включая 80)
- **Низкая оценка** — от 0 до 30 (не включая 30)

#### Топ-7 платформ по количеству игр:
- PS2, DS, PS3, X360, Wii, PSP, PC

## 🔍 Ключевые выводы

### Эпоха расцвета консольных платформ
✅ **Доминирование Sony и Nintendo**:
- PS2 — абсолютный лидер по количеству игр (1,972)
- DS и PS3 формируют ядро рынка 7-го поколения

✅ **Переход к HD-графике**:
- Появление PS3 и X360 изменило стандарты качества
- ПК сохраняет стабильную позицию в индустрии

### Особенности оценок
✅ **Высокая доля отсутствующих оценок**:
- ~50% игр не имеют пользовательских оценок
- ~56% игр не имеют оценок критиков

✅ **Баланс качества**:
- Большинство игр получают средние оценки
- Высокие оценки получают лишь 5-9% проектов
